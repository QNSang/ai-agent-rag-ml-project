# AI Agent with Retrieval - Augmented Generation (RAG)

## Mục tiêu
Dự án xây dựng một hệ thống Agent sử dụng LLM kết hợp với kỹ thuật RAG để truy xuất và sinh câu trả lời dựa trên một tập hợp tài liệu cụ thể.
